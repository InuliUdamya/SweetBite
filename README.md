# SweetBite – E-Commerce Cupcake Website

SweetBite is a fully responsive e-commerce bakery website designed to showcase cupcake products, allow users to add items to a cart, and contact the bakery through a backend-connected form.
This project demonstrates real-world e-commerce functionality with clean UI, animations, and a working database integration.

## Overview

SweetBite is a multi-page *e-commerce website* built to provide an interactive shopping experience.
It allows customers to browse products, manage a cart, update quantities, and send messages through a functional backend form.
Designed with modern UI principles, animations, and mobile responsiveness.

##Key Features

**E-Commerce Functionality**
 - Add items to cart
 - Increase/decrease quantities
 - Automatic total calculation

**Responsive Frontend** using HTML, CSS, JavaScript, Bootstrap

**Animated Hero Pages**

**Contact Page with Backend + Database**
 - Stores messages in a database
 - Requires proper DB setup

**Footer Search Bar** on all pages

**Smooth, modern user experience**

## Database Requirements
The contact page will only work properly if connected to a database.

1. create a MySQL database 
2. Import `database.sql` (incleded)
3. Update `connect.php` with your database credentials:
   $user = "your_db_user";
   $pass = "your_db_password";
   $dbname = "db_name";
4. Open the HTML form in browser
